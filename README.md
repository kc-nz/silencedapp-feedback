# Silenced — Feedback & Issues

This is the public feedback repository for [Silenced](https://silenced.app), an end-to-end encrypted communication platform.

## How to Submit Feedback

Head to the [Issues](https://github.com/kc-nz/silencedapp-feedback/issues) tab and create a new issue. Please use one of the following labels to help us categorize your submission:

- **bug** — Something isn't working as expected
- **feature request** — An idea or suggestion for new functionality
- **feedback** — General thoughts, impressions, or usability comments

## Before Posting

- Search existing issues to avoid duplicates
- Include steps to reproduce if reporting a bug
- One issue per topic — keep things focused

## What Happens Next

We review all submissions regularly. Popular requests and confirmed bugs will be prioritized accordingly. We may follow up in the issue thread if we need more details.
